The first "P" in PHP stands for "PHP: Hypertext Preprocessor."

It's a recursive acronym, where the "P" in PHP stands for PHP itself.

So, PHP originally stood for "Personal Home Page," but it was later changed to "Hypertext Preprocessor"

to reflect its expanded capabilities beyond personal home pages to include dynamic web page generation and server-side scripting.


PHPの最初の「P」は「PHP: Hypertext Preprocessor」の略です。

これは再帰的な頭字語で、PHPの「P」はPHP自体を指します。つまり、PHPはもともと「Personal Home Page」の略でしたが、

後に「Hypertext Preprocessor」と変更されました。これは、個人のホームページから始まり、

動的なウェブページの生成やサーバーサイドのスクリプティングなど、より幅広い機能を反映するためです。

It's a high-level programming language named after the comedy troupe Monty Python.

Guido van Rossum, the creator of Python, named it after the BBC television series "Monty Python's Flying Circus."

He wanted to choose a name that was short, unique, and slightly mysterious.


Pythonは、コメディトループMonty Pythonにちなんで名付けられた高水準プログラミング言語です。

Pythonの創始者であるGuido van Rossumは、BBCテレビシリーズ「Monty Python's Flying Circus」にちなんで名前を付けました。

彼は、短くてユニークで、少し神秘的な名前を選びたかったのです。その結果、Pythonは頭字語ではなく、言語の名前です。
