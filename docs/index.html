The JavaScript library that turns complicated code into something more restful.

Potato JavaScript is another way of coding for web applications.
It extends HTML code with custom tags and integrates with other libraries like Chart.js, jQuery, and AngularJS.
You can create objects such as buttons, dropdown lists, text boxes, tables, menus, charts, and more
by providing the relevant attribute name and value inside an HTML tag, which allows you to write less code and keep it simple, saving time when building things.

It is easy to learn, use, and master.

For those who code with AngularJS, they can use ng-code inside the PotatoJS tag as well.
For example, you can put ng-model inside the tag and bind it to other objects using {{data}}.


複雑なコードをよりシンプルなものに変えるJavaScriptライブラリです。
ポテトJavaScriptは、Webアプリケーションのコーディングの別の方法です。
カスタムタグを使用してHTMLコードを拡張し、Chart.js、jQuery、AngularJSなどの他のライブラリと統合します。
HTMLタグ内に関連する属性名と値を指定することで、ボタン、ドロップダウンリスト、テキストボックス、テーブル、メニュー、チャートなどのオブジェクトを作成でき、
コードを少なくしてシンプルに保ち、物事を構築する際に時間を節約できます。

習得、使用、マスターするのが簡単です。

AngularJSでコーディングする人々にとっては、PotatoJSタグ内でもng-codeを使用できます。
例えば、タグ内にng-modelを置いて、{{data}}を使って他のオブジェクトにバインドすることができます。

The basic syntax of the code looks like this:

コードの基本的な構文は次のようになります。

<[system name]-[object type]-[object id] [attribute1 attribute2...] />

◎ Potato's Objects: (各オブジェクトコードの基本的な構文)

  • ■ POTATO-BUTTON

    <potato-button-buttonID

    value="Click Me"

    background="yellow"

    marginTop="10px"

    image="image/img-p.png 21 21"

    onclick="alert('I am OButton')"

    /potato-button-buttonID>

  • ■ POTATO-CALENDAR

    <potato-calendar-calendarID

    margintop="-15px"

    value="potato.today()"

    /potato-calendar-calendarID>

  • ■ POTATO-CHART

    <potato-chart-chartID

    chartType=Bar

    stacked=true

    data='[
    ["Menu1","1","3","4","8"],
    ["Menu2","2","5","7","14"],
    ["Menu3","3","1","4","8"],
    ["Menu4","5","7","12","24"]
    ]'

    border="0px solid black"

    colorIndex="#e3ff7f,#a1a4ff,yellow,orange,green,grey"

    xLabelData=menu1,menu2,menu3,menu4

    xLabelAngle=10

    y2AxesLabel="calories"

    data2Label="calories"

    dataLabel="Protein,Fat,Sugar"

    labelFontSize=18

    label2FontSize=18

    legendDisplay=true

    legendPosition="bottom"

    pointBorderColorIndex="red,blue,yellow,orange,green,grey"

    pointBorderWidth="2,2,2"

    pointStyle="rect,rectRot,triangle"

    title="my OChart BAR"

    titleDisplay=true

    height=330px

    width=500px

    click='return alert(potato.chartYvalue)'

    /potato-chart-chartID>
